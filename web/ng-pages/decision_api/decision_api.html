Api

На цій сторінці описані методи, які можна використовувати через API, вимоги до формату даних, що передаються та приклади результатів. Для використання API потріно вказати в заголовках запиту API-ключ користувача, від імені якого здійснюються дії, і, файл активної сесії Jira.

API-ключ та файл сесії використовуються для ідентифікації користувача та отримання даних про задачі із Jira. Якщо дані про задачу вказані в запиті, заголовки для ідентифікації не потрібні.

Якщо Ви вже авторизовані в системі, ідентифікаційні дані вводити не потрібно.

Зміст:

  • Визначення частотної характеристики мови проекту
  • Аналіз якості опису задачі
  • Класифікація задач по якості опису
  • Класифікація задач за пріоритетом виконання
  • Класифікація задач за посиланнями на зовнішні ресурси
  • Кластеризація задач за описом
  • Пошук подібних задач до заданої
