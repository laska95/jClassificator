Api

На цій сторінці описані функції для аналізу задач, які можна викликати із зовнішніх ресурсів, вимоги до формату даних, що передаються та приклади результатів. Для використання API потріно вказати api-ключ користувача, від імені якого здійснюються дії, і, якщо потрібно, передати в заголовках запиту ключ сесії від Jira.

Зміст:

  • Визначення параметрів мови проекту
  • Аналіз якості опису задачі
  • Перевірка задачі на наявність коректного опису
  • Класифікація за пріоритетом
  • Кластеризація за спільними ресурсами
  • Кластеризація за описом
  • Пошук подібних задач до даної
